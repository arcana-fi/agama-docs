# V2 — Lending Market + Looping

Condition: a native lending market deploys on the Rayls Public Chain.

## What V2 adds

**Cork Pool agaINV/USDXP**

Resolves the duration risk that blocks RWA looping. LPs deposit USDXP into the Cork Pool, which mints cPT (principal token) and cST (swap token). cST gives the right to swap agaINV for USDXP instantly, regardless of redemption queue. This makes agaINV acceptable as collateral on a lending market. Bootstrap via LP incentive program.

**Protected Loop Vault agaINV — Scenario A**

| Metric | Value |
|--------|-------|
| Deposit | 100,000 USDXP |
| Flow | Deposit → agaINV → collateral on lending (Cork protects) → borrow USDXP 70% LTV → re-deposit → 2-3 cycles |
| Effective exposure | ~230,000 USDXP for 100,000 initial |
| Net yield target | ~20% APY |
| Requirement | Rayls native lending market |

**agaYLD looping via Arbitrum — Scenario B**

Available before Scenario A (does not require Rayls lending market). agaYLD contains public assets (BUIDL + USDY + TBILL + USDXP) — no privacy required. Bridge native Rayls → Arbitrum (Orbit). Collateral on Morpho Arbitrum. Borrow USDC. Bridge back. Re-deposit. Net yield target: 12–15% APY. Arbitrum grants available.

**USDXP basis trade — Scenario C**

agaINV 12% minus USDXP borrow 5.5% = 6.5% spread × 2.5x leverage. Net yield target: 15–18% APY. Requires Rayls lending market.
